In this Project I Have Created Real World Backend RESTful API for E-Commerce Platform Ready for mobile or Web E-Shop Application

Following are the APIs with their functionality -

//Products APIs
(GET) localhost:3000/api/v1/products  - Return list of all product in json format
(POST) localhost:3000/api/v1/products  - Add new Product into Database and return added product data
(PUT) localhost:3000/api/v1/products/:productID  - Update product with givn ID and return updated data 
(DELETE) localhost:3000/api/v1/products/:productID  - Delete product with givn ID and return deleted Product Data

//Category APIs
(GET) localhost:3000/api/v1/category  - Return list of all categories in json format
(POST) localhost:3000/api/v1/category  -  Add new category into Database and return added Category data
(PUT) localhost:3000/api/v1/category/:categoryID  -  Update category with given ID and return updated data
(DELETE) localhost:3000/api/v1/category/:categoryID  -  Delete category with givn ID and return deleted category Data
